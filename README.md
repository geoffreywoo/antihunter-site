# antihunter.com

Astro + Tailwind landing page for AntiHunter.

## Dev

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy (Vercel)

1) Push this repo to GitHub.
2) Import into Vercel.
3) Set the project domain to `antihunter.com`.
4) Update DNS per Vercel instructions.
